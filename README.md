# Metamas - Sitio web (estático)

## Cómo abrir
- Abrí `index.html` con doble click, o servilo con un servidor local.

## Deploy rápido
- GitHub Pages: subir esta carpeta como repo y activar Pages (branch main / root).
- Cloudflare Pages: importar repo y build command vacío (sitio estático).
